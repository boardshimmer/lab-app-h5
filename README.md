# lab-app-h5

## 项目简介

`lab-app-h5` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅作为实验性、轻量级的静态页面集合，方便快速访问与分享。

## 目录说明

```
lab-app-h5/
├── index.html          # 入口页面（可选）
├── pages/              # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 公共资源（样式、脚本、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本说明文件
```

- `pages/`：每个独立页面以 `.html` 文件形式存放，文件名应与内容对应。
- `assets/`：集中存放页面共用的静态资源，便于统一管理。

## 页面归档说明

各 HTML 页面多为独立、自包含的演示或工具页面，可直接通过浏览器打开运行。  
页面之间无强依赖关系，可根据需要单独使用或归档保存。

如需添加新页面，请：

1. 在 `pages/` 目录下创建新的 `.html` 文件。
2. 如页面需要额外资源（样式、脚本、图片），请放置于 `assets/` 对应子目录中。
3. 保持文件命名清晰、简洁，避免特殊字符。

## 维护说明

- 本仓库主要用于个人或团队内部归档，不设严格的发布流程。
- 页面内容可能随时更新、删除或替换，不保证向后兼容。
- 如有疑问或建议，欢迎提交 Issue 或 Pull Request。
